
# 隐私政策

**生效日期和最后更新日期：** 2026年2月25日

NimoteCode（以下简称“我们”、“本公司”）运营 NimoteCode 移动应用（以下简称“应用”或“服务”），一款面向 iOS 和 Android 的专业跨平台远程开发 IDE。我们致力于以透明、安全、负责的方式保护您的隐私并处理个人信息。

本隐私政策描述了您下载、安装、注册或使用 NimoteCode 应用时，我们如何收集、使用、披露、存储和保护您的信息。使用应用即表示您同意本政策所述实践。如不同意，请勿使用服务。

我们强烈建议您仔细阅读本政策。如有任何问题，请联系 aoun.ma@foxmail.com。

## 1. 我们收集的信息

### 1.1 您直接提供的信息
- **账户信息**：电子邮件地址（用于注册、登录、密码重置和订阅管理）；Google 账户标识符和基本个人信息（若选择通过 Firebase Authentication 的 Google 登录）。
- **支持和沟通**：您主动提供的信息，例如通过电子邮件或应用内反馈联系支持时，包括问题描述、截图或附件日志。
- **订阅和支付数据**：订阅层级选择和交易标识符。所有支付处理（包括信用卡详细信息）均由 Apple App Store 或 Google Play Billing 独家处理，订阅管理由 RevenueCat 协助——我们**不**收集、存储、访问或处理完整的支付卡号、CVV 或账单地址。

### 1.2 自动收集的信息
- **设备和使用数据**：设备型号、制造商、操作系统版本、应用版本、语言设置、屏幕分辨率、时区、唯一设备标识符（例如，若允许则为 IDFA/GAID）、崩溃报告、性能指标和错误日志（匿名化）。
- **分析数据**：聚合的、匿名化的使用统计数据（例如，使用哪些功能、会话时长、SSH 连接或 Git 操作的频率），用于改进应用。此项为可选，并在设置中可禁用。我们为此目的使用 Firebase Analytics。
- **连接元数据**：SSH/SFTP 会话的时间戳、IP 地址（临时，不长期记录）和连接状态（仅用于调试和防止滥用——**不记录命令内容或文件内容**）。

### 1.3 我们明确不收集的信息
我们**不**收集、记录、传输至我们的服务器或存储：
- SSH/SFTP 凭据（私钥、密码、密码短语、令牌或主机密钥）——这些仅在您的设备上使用平台安全机制（iOS Keychain / Android Keystore 加密）**本地存储**。
- SSH 终端中执行的任何命令内容。
- 通过 SFTP 或内置编辑器浏览、查看或编辑的文件内容。
- 源代码、仓库内容或项目数据（除编辑器功能的临时缓存外，并在会话结束或应用关闭时清除）。
- 使用用户配置模型时的 AI 相关数据：提示、代码上下文、响应或输出直接从您的设备发送至第三方 AI 提供商（例如 OpenAI、Anthropic、本地 LLM）——我们**不**拦截、代理、记录或存储这些交互。

## 2. 我们如何使用您的信息

我们仅将收集的信息用于以下合法目的：
- 提供、操作、维护、安全、个性化并改进应用及其功能（例如 SSH 连接、代码高亮、Git 可视化、AI 辅助设置）。
- 通过 Firebase Authentication 验证您的身份、管理您的账户，并通过 RevenueCat 启用基于订阅的功能。
- 通过平台计费 API 和 RevenueCat 处理和验证应用内购买及订阅状态。
- 回复支持查询、发送服务通知（例如更新、订阅提醒）并传达重要变更。
- 检测、防止和调查技术问题、滥用、欺诈或未授权访问。
- 使用 Firebase 生成匿名化、聚合分析数据，用于产品开发和性能优化。

## 3. 数据共享和披露

我们**不**将您的个人信息出售给第三方。

我们仅在以下有限情况下共享信息：
- 与 Apple Inc. 和 Google LLC 共享，以满足应用分发、计费、崩溃报告（若启用）和分析要求。
- 与受信任的服务提供商共享，例如 Firebase（用于身份验证和分析）和 RevenueCat（用于订阅管理），这些提供商受严格的数据处理协议约束，要求提供等效保护。
- 根据法律、法规、传票、法院命令或政府要求共享。
- 在合并、收购、重组合并或资产出售相关情况下共享——我们将在法律要求的情况下提供通知并获得同意。

**AI 功能重要说明**：当您配置外部 AI 模型或 API（第三方或本地）时，您的代码片段、提示和上下文直接传输至该提供商的服务器。我们**不**接收、存储或处理此数据。您有责任审查并遵守您选择集成的任何第三方 AI 服务的隐私政策。

## 4. 数据存储、安全和保留

- **本地存储**：SSH 凭据、应用首选项和临时编辑器缓存使用平台原生安全存储在设备上加密。
- **服务器端存储**：最小账户数据（电子邮件、订阅状态）存储在安全的、加密的云基础设施中，具有访问控制和定期安全审计，通过 Firebase 和 RevenueCat 管理。
- **保留**：我们仅在概述的目的所需时间内保留个人信息（通常为账户生命周期 + 法律合规的删除后 1 年），或根据法律要求。匿名化分析数据可能保留更长时间。

我们实施合理的技术、管理和物理保障措施来保护您的数据，尽管没有系统是 100% 安全的。

## 5. 国际数据传输

您的信息可能被传输至您所在国家以外的司法管辖区（例如美国、欧盟或亚洲的云服务器）并在那里处理。我们使用适当的保障措施（例如适用的标准合同条款）来确保保护符合本政策和适用法律。

## 6. 您的权利和选择

根据您的位置和适用法律（GDPR、CCPA/CPRA 等），您可能享有以下权利：
- 访问、更正、删除或移植您的个人信息。
- 选择退出某些处理（例如分析）。
- 在处理基于同意的情况下撤回同意。
- 向监督机构提出投诉。

要行使这些权利或请求删除您的账户/数据，请发送电子邮件至 aoun.ma@foxmail.com。我们将在法律要求的时间范围内（例如 30–45 天）响应。

您还可以在应用内管理首选项（例如禁用分析）。

## 7. 儿童隐私

应用不针对 13 岁以下儿童（或您所在司法管辖区的法定最低年龄）。我们不会故意收集儿童的个人信息。如果我们意识到此类收集，我们将立即删除。

## 8. 本隐私政策的变更

我们可能更新本政策以反映实践、技术或法律变更。我们将通过应用内通知、电子邮件或 App Store 更新说明通知您重大变更。变更后的继续使用即构成接受。
